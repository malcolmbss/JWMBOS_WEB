LifeFamily Content Resources Project Prototype

Note: If you are looking for the LifeFamily Church web site... this is not it. This is a test site prototype for possible future updates to the web site. The official LifeFamily website is https://Life.Family

Project Mission, Objectives, Caveats, & Disclaimers

  • The objective is not to design a new web site and/or mobile app. The objective is to create a framework for storing, accessing, retrieving, & notifying users with information related to all events, past and future, for LifeFamily Church.
  • Where a new/modified web site might result down the road, again, that is not in the current scope of this project.
  • The prototype website is not real. Other than specific items on the pages specifically related to the prototype work, the remainder is a 'movie-set' representing the real Life.Family site. There are items missing. Ignore it.
  • The UI/UX of the new function is irrelevant to the design. The purpose of the web presentation is not to show how it should appear, rather to show what this new function can do. Signicant additional UI/UX design will occur later in the process to determine how all of this will look and act.
  • The new navigation tree on the left is a suggestion. We are not proposing a final complete tree structure. The design is a framework. The design intent is that authorized admin users can create / delete / move nodes in the tree.
  • Content for each node will be generated by the individual ministries.
  • The current tree shows what 'could' be. Once in production, empty nodes will not appear. For example, if one campus does not have any content for LifeMen ministry, that node will not appear for that campus.
  • Security roles will also protect certain sensitive or private nodes/content.
  • Every node in the entire navigation tree 'can' have it's own separate subscribable calendar. Calendar events in lower nodes can be shared with other nodes by the calendar event creator.
  • When a user subscribes to a calendar on their mobile device or google calendar, all subsequent updates to the calendar will automatically appear on the user's mobile device calendar or google calendar.
  • In the current prototype, only "Southwest Campus LifeMen" is active on the side bar Navigation tree. To view LifeMen page:
    • Go to the prototype using the button below
    • Scroll down the new Site Navigation side bar
    • Navigate to & click: LifeFamily→Campuses→Austin (Southwest-Main)→Ministries→LifeMen
    • The LifeMen calendar is REAL You can really subscribe to this calendar using instructions in the popup (although the calendar is not yet officially representing the Southwest Campus LifeMen ministry)
    • Only admins can add events to the calendar. But added events should appear on all subscribed devices.
  • In the current prototype, the search bar is not functional.
